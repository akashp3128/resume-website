Cloudflare Worker Deployment Guide

Setup Instructions

  1. Install Wrangler (Cloudflare CLI): 
    npm install -g wrangler
  2. Log in to your Cloudflare account: 
    wrangler login
  3. Navigate to the worker directory and deploy: 
    cd cloudflare-worker
wrangler deploy
  4. Set up Public Access for your R2 bucket:
    • Go to Cloudflare dashboard → R2
    • Select your "resume" bucket
    • Go to Settings → Public Access
    • Enable public access and set up a custom domain (e.g., assets.akashpatelresume.us)

Note: Files are stored in your R2 bucket with organized paths:

  • resume/2023-04-11T12-34-56-789Z-filename.pdf - Resume files
  • eval/2023-04-11T12-34-56-789Z-document.pdf - Evaluation documents
  • photo/2023-04-11T12-34-56-789Z-image.jpg - Photos
